# Backed
This project is a game backlog management web application that will allow users to organize and track the video games they want to play, are currently playing, or have completed.
The backend will be built with Spring Boot as a way to continue improving my backend development skills and deepen my experience with the Spring ecosystem.
The frontend will be developed with React to practice and gain more experience with modern frontend development.
The goal of this project is to strengthen my full-stack development skills by combining a Java/Spring backend with a React frontend in a practical, real-world use case.
